## Manual Install
Chrome
1. Download source zip, unzip into folder
2. Go to `chrome://extensions`
3. Enable developer mode
4. Load unpacked, select the folder

Firefox
1. Download source zip, unzip into folder
2. Go to `about:debugging#/runtime/this-firefox`
3. Click `Load Temporary Add-onâ€¦`
4. Select the folder
